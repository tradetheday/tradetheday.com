## TECHNICAL SEO REQUIREMENTS

### Every Page Must Have:
- [ ] Unique title tag (60 chars max, keyword first)
- [ ] Meta description (155 chars, includes CTA)
- [ ] H1 with primary keyword
- [ ] Schema markup (appropriate type)
- [ ] Canonical URL
- [ ] Open Graph tags
- [ ] Internal links to money pages
- [ ] Breadcrumb navigation

### Site-Wide:
- [ ] XML sitemap at `/sitemap.xml`
- [ ] Robots.txt at `/robots.txt`
- [ ] 301 redirects for old URLs
- [ ] HTTPS everywhere
- [ ] Mobile responsive
- [ ] Core Web Vitals optimized
- [ ] Structured data for Organization
- [ ] hreflang tags on regional pages
- [ ] Language selector in header/footer
