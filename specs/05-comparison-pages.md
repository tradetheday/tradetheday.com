# JTBD: Comparison Pages

## Job Statement
**When** a trader is deciding between specific brokers/exchanges/prop firms,
**I want to** provide detailed head-to-head comparisons,
**So that** they click through to sign up with one (or both).

## Success Criteria
- [ ] "[Broker A] vs [Broker B]" pages for top combinations
- [ ] Interactive comparison tables
- [ ] Clear winner declarations
- [ ] Both affiliate links prominent

## Standard Comparison Structure

```
1. Hero Section
   - "[A] vs [B]: Which is Better in 2026?"
   - Quick verdict badge
   - Both logos

2. Quick Comparison Table
   - Side-by-side key stats
   - Winner indicators per category

3. Overview Comparison
   - Brief intro to each
   - Key differences

4. Detailed Category Comparisons
   - Fees (with tables)
   - Platforms
   - Instruments
   - Regulation
   - Customer service
   - Each with winner declaration

5. Best For Section
   - "[A] is better for..."
   - "[B] is better for..."

6. FAQ Section
   - "Is [A] better than [B]?"
   - "Which has lower fees?"
   - etc.

7. Final Verdict
   - Overall winner
   - Nuanced recommendation

8. Dual CTA
   - Both signup buttons
   - Partner codes if available
```

## SEO Requirements
- Title: "[A] vs [B] 2026: Fees, Features & Which is Better"
- URL: `/compare/[a]-vs-[b]/`
- Schema: FAQ, potentially ComparisonTable
- Target "[a] vs [b]" exact match keywords
